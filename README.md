# 欢迎来到mimo!

mimo是一款完全去中心化的协议，具有自动化流动性，旨在为 DeFi 的未来生态提供动力。在 IoTeX 艾坦星区块链的支持下，mimo 利用智能合约和自动做市 \(AMM\) 算法来管理流动性池并连接加密资产的买家/卖家。 

mimo 协议的第一个应用实例是 mimo.exchange——这是一个去中心化交易所（DEX），用于 ERC20 和 XRC20 等资产的点对点交易。
